# Terraform Infrastructure

This directory contains the Infrastructure as Code (IaC) used to provision the CloudSphere AI Platform.

## Planned Infrastructure

- AWS VPC
- Public and Private Subnets
- Internet Gateway
- NAT Gateway
- Security Groups
- IAM Roles and Policies
- Amazon EKS Cluster
- Managed Node Groups
- Application Load Balancer
- Route53
- ACM Certificates
- Amazon ECR
- CloudWatch
- S3 Backend
- DynamoDB State Locking

## Deployment

Terraform will provision all AWS infrastructure required to deploy the Kubernetes platform.
